Why "the curry coder"?

Good curry and good backend code have a lot in common — the right ingredients, patience under heat, and no shortcuts that come back to bite you. That's how I write software.

By day I work at TrilaSoft Solutions on RedSky Mobility (CMMI Level 3), where I've handled everything from a full Spring 3 → 5 and MySQL 5 → 8 migration to solely rewriting a Microsoft Outlook plugin from AWS Lambda to JAX-RS.

  • Migrations that don't break production
  • Boring, reliable, well-tested code
