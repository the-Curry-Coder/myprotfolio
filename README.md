# 🍛 the_curry_coder — Portfolio

Personal portfolio of **Ujjwal Kumar Singh**, Java Backend Developer.

**Live:** https://the-curry-coder.github.io/myprotfolio/

## About

A single-page, dependency-free portfolio with a warm "curry" theme — because good backend code, like good curry, needs the right ingredients and patience under heat.

**Highlights:**
- ⚡ Interactive hero with spice-particle canvas that reacts to your cursor
- 🔄 Rotating headline, count-up stats, scrolling tech ticker
- 🃏 3D-tilt project cards and magnetic buttons
- ♿ Respects `prefers-reduced-motion` and touch devices
- 📦 Zero JS libraries — one HTML file, vanilla CSS/JS

## Tech

Plain HTML + CSS + vanilla JavaScript. Google Fonts (Fraunces, Inter, JetBrains Mono). That's it.

## Run locally

Just open `index.html` in a browser — no build step, no server needed.
